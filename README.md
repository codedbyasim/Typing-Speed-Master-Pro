# Typing Speed Master Pro

**Typing Speed Master Pro** is a modern, browser-based typing tutor inspired by TypingMaster. It helps users improve typing speed and accuracy through interactive modes, live feedback, and stylish UI â€” built using HTML, CSS, and JavaScript.

---

## Features

- **Login/Register System** (using Local Storage)
- **Personal Dashboard** with:
  - Avatar & welcome message
  - Typing stats (Best WPM, Accuracy, Total Tests)
  - Progress bar & motivational quote
  - Practice badges (e.g. Speedster, Accuracy Champ)
- **Typing Modes**:
  - Beginner
  - Intermediate
  - Advanced
  - Code Typing (HTML, CSS, JS)
  - Custom Text
- **Live Typing Feedback**:
  - WPM (Words Per Minute)
  - Accuracy %
  - Mistakes counter
  - Typewriter sound (optional)
